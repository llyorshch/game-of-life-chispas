# Ejercicio: Juego de la Vida de Conway en HTML y JavaScript

Este repositorio contiene un ejercicio para desarrollar el [Juego de la Vida de Conway](https://es.wikipedia.org/wiki/Juego_de_la_vida) utilizando HTML y JavaScript.  
El Juego de la Vida es un autómata celular ideado por el matemático británico John Horton Conway. Puedes encontrar más información en la [Wikipedia en inglés](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

Este ejercicio se realizó con motivo del evento **Code Retreat 2025** celebrado en [Flywire](https://www.flywire.com/) en sus oficinas de Valencia, organizado por la comunidad [DevScola](https://devscola.org/).
